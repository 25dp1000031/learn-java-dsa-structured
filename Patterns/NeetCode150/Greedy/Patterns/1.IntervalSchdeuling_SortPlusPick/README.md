🔑 1. Sort + Pick (THE most important)

If you learn only one thing, make it this.

Idea:
Sort the input, then keep picking the “best” option greedily.

Covers:

Activity selection (sort by end time)
Interval merging / non-overlapping intervals
Job sequencing
Assign cookies

👉 If you see:

intervals
pairs
scheduling

➡️ Think: “sort first, greedy later”
